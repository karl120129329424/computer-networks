# UDP Echo Lab

UDP эхо-сервер и клиент.

### Лабораторная работа 1: Основы UDP. Эхо-сервер и клиент

Цель: Освоить основы работы с UDP-сокетами, создать простейший сервер, который возвращает полученное сообщение обратно клиенту.

Задание:
1. Написать UDP-сервер, который:
    - Принимает датаграммы от клиента.
    - Выводит полученное сообщение и адрес клиента в консоль.
    - Отправляет это же сообщение ("эхо") обратно клиенту.
2. Написать UDP-клиент, который:
    - Считывает строку с клавиатуры.
    - Отправляет её серверу.
    - Ожидает и выводит ответ от сервера.

Теоретическая часть: Сокеты, socket(), bind(), sendto(), recvfrom(), sockaddr_in.
